ofxDaveOpenCV
=============

Just a simple class that does all the openCV Things in smaller blocks
Specifically for use within Playable City Project: Shadowing

## Requirements

ofxOpenCV
ofxCalibration

## API
===
// Setup 

<code>
void setup(int width, int height, int framerate);
</code>

</code>
